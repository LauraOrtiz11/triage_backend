# Backend - Sistema de Triage Inteligente 

Este repositorio contiene el **backend** del sistema de triage clínico, desarrollado en **.NET Core**.  
El sistema gestiona el flujo de atención de pacientes en procesos de triage, consultas médicas y administración de usuarios.

---

##  Funcionalidades principales

- Gestión de **usuarios** (administrador, médico, enfermero, paciente).
- Registro y actualización de **pacientes**.
- Módulo de **triage**: registro de síntomas, signos vitales y prioridad.
- Administración de **historias clínicas** (consultas, diagnósticos, tratamientos).
- **Reportes y estadísticas** para evaluar la eficiencia del sistema.
- Seguridad y validaciones de acceso según el rol del usuario.

---

## Tecnologías utilizadas

- **.NET Core** (API REST)
- **SQL Server**
- **Swagger** (documentación de endpoints)
